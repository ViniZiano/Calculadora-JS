# 📱 Calculadora Web

Uma calculadora moderna desenvolvida com **JavaScript, HTML e CSS**, com foco em interface elegante, boas práticas e interações suaves.

---

## 🚀 Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão  
- Tratamento de erros (ex: divisão por zero)  
- Suporte ao teclado  
- Interface moderna e responsiva  
- Animações suaves (clique, ripple, destaque de resultado)  
- Formatação de números  
- Código organizado com **ES Modules**

## ⌨️ Atalhos do teclado

| Tecla       | Ação              |
|------------|------------------|
| 0-9        | Inserir números  |
| + - * /    | Operações        |
| Enter      | Calcular (=)     |
| Backspace  | Apagar           |
| Esc        | Limpar tudo      |
